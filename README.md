# 🏦 Binary Bank (BB) Society - v1.0

**A Secure, Modular, and Interactive Banking Management System built with Python.**

---

## Project Overview

Binary Bank (BB) is a robust Banking Application designed to demonstrate the power of **Object-Oriented Programming (OOPs)** and **Data Persistence**. It allows users to create accounts, perform secure transactions, and visualize bank analytics through a modern web interface.

## Key Features

- **Secure Account Management:** Uses Encapsulation (Private Variables) to protect sensitive user data.
- **Persistent Storage:** Integrated with **JSON** for flat-file database management.
- **Dynamic Transactions:** Real-time Deposit and Withdrawal logic with limit validations.
- **Data Analytics Dashboard:** Built-in Admin dashboard using **Pandas** and **Plotly** to visualize gender distribution and wealth metrics.
- **Modular Architecture:** Separation of Backend Logic (`main.py`) and Frontend UI (`app.py`).

## Tech Stack

- **Language:** Python 3.10+
- **Frontend:** Streamlit
- **Data Handling:** Pandas, JSON
- **Environment:** Anaconda / Virtual Environment

## 📁 File Structure

- `main.py`: The "Engine" containing Core Logic, OOPs Classes, and Database Handlers.
- `app.py`: The "Body" containing the Streamlit Web Interface.
- `data.json`: The "Database" storing user records securely.
- `reference.py`: Initial logic reference used for research and development.
